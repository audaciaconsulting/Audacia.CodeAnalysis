# Audacia.CodeAnalysis.Libraries

This project uses analyzers from the [Audacia.CodeAnalysis](https://github.com/audaciaconsulting/Audacia.CodeAnalysis/blob/master/README.md) and configures analyzer rulesets for use in code libraries.

See the [Audacia.CodeAnalysis](https://github.com/audaciaconsulting/Audacia.CodeAnalysis/blob/master/README.md) project for full information about how to use the code analysis library.
